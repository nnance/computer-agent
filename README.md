# Computer Agent

A simple computer agent using Anthropic's Claude model.  So think of it like as a coding agent but for general computer tasks.
It can read and write files, run commands, and use tools to accomplish tasks.

## Features
- Uses Anthropic's Claude model for agentic behavior and natural language understanding.
- Powerful agentic capabilities to perform complex tasks autonomously.  Potentially running for several minutes or hours.
- Very small set of run-time dependencies.
- Simple configuration and setup.
- Modular architecture for easy extension and customization.
